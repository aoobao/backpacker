# 大富翁自由行

## 代码运行
```
yarn install  // setup

yarn serve  // run

yarn build  // build
```
基于大富翁自由行实现,参考游戏玩法:

[https://j9981168.pixnet.net/blog/post/239327645](https://j9981168.pixnet.net/blog/post/239327645)
